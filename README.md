ระบบตรวจซองเติมซองอัตโนมัติผ่านช่องทาง Discord
<br>
> npm i<br>
> npm i request<br>
> <br>
> สามารถเอาคีย์ได้จากในเว็บมาใส่ที่ config.php บรรทัดที่ 3<br>

> แก้ไข Prefix ได้ที่ index.js บรรทัดที่ 5<br>
> <br>
> Prefix เริ่มต้น $<br>
